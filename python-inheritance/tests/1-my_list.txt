The ``print_sorted`` module
======================

Using ``print_sorted``
-------------------

First import ``MyList`` from the ``1-my_list`` file:

    >>> MyList = __import__('1-my_list').MyList

Tests:
    Test1 - instantiate empty list
    >>> my_list = MyList()
    >>> print(my_list)
    []

    Test2 - append and check original list
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> print(my_list)
    [1, 4, 2]

    Test3 - use print_sorted on unsorted list
    >>> my_list.print_sorted()
    [1, 2, 4]
    >>> print(my_list)
    [1, 4, 2]

    Test4 - sorted input list
    >>> my_list2 = MyList([1, 2, 3])
    >>> my_list2.print_sorted()
    [1, 2, 3]

    Test5 - unsorted input with negative numbers
    >>> my_list3 = MyList([3, -1, 2])
    >>> my_list3.print_sorted()
    [-1, 2, 3]

    Test6 - empty list
    >>> my_list4 = MyList([])
    >>> my_list4.print_sorted()
    []
