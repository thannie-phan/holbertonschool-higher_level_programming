The ``text_indentation`` module
======================

Using ``text_indentation``
-------------------

First import ``text_indentation`` from the ``5-text_indentation`` file:

    >>> text_indentation = __import__('5-text_indentation').text_indentation

Tests:
    Test1 - simple text no delimeters
    >>> text_indentation('Holberton School')
    Holberton School

    Test2 - simple text with 1 delimeters
    >>> text_indentation('Holberton.School')
    Holberton.

    School

    Test3 - text is not a string
    >>> text_indentation(5.42)
    Traceback (most recent call last):
    ...
    TypeError: text must be a string

    Test4 - text is blank
    >>> text_indentation(' ')

    Test5 - text is empty
    >>> text_indentation()
    Traceback (most recent call last):
    ...
    TypeError: text_indentation() missing 1 required positional argument: 'text'

    Test6 - text has all 3 delimeters
    >>> text_indentation('Holberton. School? How are you: John')
    Holberton.

    School?

    How are you:
    
    John
